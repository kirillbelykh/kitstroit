# KitStroit

Сайт-воронка для строительства загородных домов в Санкт-Петербурге и Ленинградской области.

Решения и правила проекта зафиксированы здесь:

- [Продукт](docs/PRODUCT.md)
- [Дизайн](docs/DESIGN.md)
- [Архитектура](docs/ARCHITECTURE.md)
- [Контент](docs/CONTENT.md)
- [Эксплуатация](docs/OPERATIONS.md)

Главный принцип первой версии: один продукт, один посетительский сценарий, два действия — позвонить или оставить заявку.

## Локальный запуск

```bash
cp .env.example .env
docker compose up --build
```

Перед запуском задайте безопасные значения пароля PostgreSQL, Argon2-хеша администратора и JWT-секрета. Telegram-токен хранится только в `.env`.
